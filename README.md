# Travel App - טיסות, מלונות וחבילות נופש

Monorepo עם אתר (Next.js) ואפליקציית מובייל (React Native / Expo) לחיפוש טיסות זולות, מלונות וחבילות נופש. שתי הפלטפורמות חולקות טיפוסים, נתוני דמו ולוגיקת API דרך `/shared`.

## מבנה הפרויקט

```
travel-app/
  shared/   - טיפוסים, קבועים (יעדים/חברות תעופה), שירותי API ונתוני mock - משותף לשתי הפלטפורמות
  web/      - Next.js 14 (App Router) + TypeScript + Tailwind CSS
  mobile/   - React Native (Expo) + TypeScript
```

## דרישות מוקדמות

- Node.js 18+ ו-npm
- לאפליקציית המובייל: אפליקציית **Expo Go** בטלפון (או אמולטור Android/iOS)

## התקנה

```bash
cd travel-app
npm install                # מתקין את התלויות של shared + web (npm workspaces)
cd mobile && npm install   # מתקין את תלויות המובייל בנפרד (כדי לא להתנגש עם Metro)
cd ..
```

## חיבור ל-APIs אמיתיים (אופציונלי)

בברירת מחדל, האפליקציה עובדת **ללא שום מפתח API** - היא נופלת אוטומטית לנתוני הדגמה (mock) עשירים ומציאותיים (יעדים פופולריים, מחירים, טיסות ומלונות). זה מאפשר להריץ ולהתנסות באפליקציה מיד.

כדי לחבר נתונים חיים:

1. הירשמו בחינם ל-[Amadeus for Developers](https://developers.amadeus.com/) וקבלו `Client ID` ו-`Client Secret` (Test tier - מספיק לפיתוח).
2. **לאתר**: העתיקו `web/.env.example` ל-`web/.env.local` ומלאו את המפתחות.
3. **למובייל**: העתיקו `mobile/.env.example` ל-`mobile/.env` ומלאו את המפתחות (עם קידומת `EXPO_PUBLIC_`).

> ⚠️ **הערה לגבי אבטחה**: משתני `EXPO_PUBLIC_*` מוטמעים בבאנדל הקליינט של האפליקציה וניתנים לחילוץ. זה סביר לשימוש אישי/פיתוח עם מפתחות Test, אך **לא מומלץ** להטמיע מפתחות production כך. לשימוש production, מומלץ שהמובייל יקרא ל-API Routes של Next.js (כמו האתר) במקום לפנות ישירות ל-Amadeus.

כל קריאת API כוללת טיפול שגיאות ונפילה אוטומטית חזרה לנתוני mock אם הקריאה נכשלת או שאין מפתחות מוגדרים. תגית "נתונים חיים" / "נתוני הדגמה" מוצגת בתוצאות כך שתמיד ברור מאיפה המידע מגיע.

ניתן להחליף ספק מלונות (Booking.com / Hotels.com / Skyscanner) על ידי מימוש פונקציה חדשה ב-`shared/src/services/api/hotelService.ts` בדומה למימוש הקיים של Amadeus.

## הרצת האתר (Web)

```bash
npm run web:dev
```

האתר יעלה בכתובת http://localhost:3000

## הרצת אפליקציית המובייל

```bash
cd mobile
npm start
```

זה יפתח את Expo Dev Tools. סרקו את קוד ה-QR עם אפליקציית Expo Go (Android/iOS), או הריצו `npm run android` / `npm run ios` להרצה באמולטור.

## פיצ'רים

- **חיפוש טיסות** - סינון לפי תאריכים, יעד, מחיר מקסימלי, מספר עצירות וחברת תעופה
- **חיפוש מלונות** - בארץ ובחו"ל, סינון לפי יעד, תאריכים, תקציב ודירוג כוכבים
- **חבילות נופש** - טיסה + מלון במחיר מאוחד עם הנחת חבילה, והשוואה בין עד 3 חבילות
- **RTL מלא** ותמיכה בעברית בשני הפלטפורמות
- נפילה אוטומטית לנתוני הדגמה עשירים כשאין חיבור API חי

## סקריפטים שימושיים (מהתיקייה הראשית)

| פקודה | תיאור |
|---|---|
| `npm run web:dev` | הרצת שרת הפיתוח של Next.js |
| `npm run web:build` | בניית גרסת production של האתר |
| `npm run shared:build` | קומפילציית חבילת shared בלבד |
| `npm run mobile:start` | הרצת Expo (חלופה ל-`cd mobile && npm start`) |
